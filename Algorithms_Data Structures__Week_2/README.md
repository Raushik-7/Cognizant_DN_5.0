
# Algorithms_Data_Structures
This repository contains solutions for Week-2 exercises as part of the Cognizant learning program.

## Exercise 1: Singleton Pattern

### Objective
Implement Linear Search and Binary Search to find products in an e-commerce platform and compare their efficiency.

### Concepts Used
- Arrays
- Linear Search
- Binary Search
- Sorting
- Time Complexity Analysis

### Features
- Store product details using a Product class.
- Search products using Linear Search.
- Search products using Binary Search.
- Compare search performance.

### Time Complexity

| Algorithm | Best Case | Average Case | Worst Case |
|------------|-----------|-------------|------------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

### Conclusion
Binary Search is more efficient for large datasets because it reduces the search space by half in every iteration. However, it requires the data to be sorted before searching.


# Design Patterns and Principles

This repository contains solutions for Design Patterns and Principles exercises as part of the Cognizant learning program.

## Exercise 1: Singleton Pattern

### Objective
Implement the Singleton Design Pattern to ensure that only one instance of a class exists throughout the application.

### Concepts Covered
- Singleton Pattern
- Private Constructor
- Static Members
- Controlled Object Creation

### Learning Outcome
Learn how to create and manage a single shared instance of a class.

---

## Exercise 2: Factory Method Pattern

### Objective
Implement the Factory Method Pattern to create objects without exposing the object creation logic to the client.

### Concepts Covered
- Factory Method Pattern
- Abstraction
- Polymorphism
- Loose Coupling

### Learning Outcome
Learn how to delegate object creation to factory classes, improving flexibility and maintainability.
